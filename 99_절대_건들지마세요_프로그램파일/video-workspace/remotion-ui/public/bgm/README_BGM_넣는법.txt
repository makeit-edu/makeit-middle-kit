[ BGM(배경음악) 넣는 방법 ]

1. 이 폴더(public/bgm)에 음원 파일(mp3, m4a, wav)을 넣습니다.
   - 반드시 "CC0" 또는 "저작자표시 불필요 + 상업적 사용 가능" 음원만 넣으세요.
     (1000명 이상이 함께 쓰므로 저작권이 깨끗한 곡만 안전합니다.)

2. 추천 무료 음원 출처 (저작자 표시 없이 상업적 사용 가능):
   - Pixabay Music : https://pixabay.com/music/   (Pixabay License, 표시 불필요)
   - YouTube 오디오 보관함 : https://studio.youtube.com  ("저작자 표시 필요 없음" 필터)
   - Free Music Archive의 CC0 태그 : https://freemusicarchive.org
   ※ Kevin MacLeod(incompetech) 곡은 "저작자 표시 필요(CC-BY)"라 배포용으로는 권장하지 않습니다.

3. 곡을 넣은 뒤 bgm-list.json 파일을 열어 tracks에
   { "file": "실제파일명.mp3", "title": "보기 좋은 제목" } 형태로 적으면
   편집기 BGM 드롭다운에 그 제목으로 나타납니다.
   (bgm-list.json에 안 적어도 파일은 파일명 그대로 목록에 나옵니다.)

4. 06 영상편집기를 열고 BGM 트랙을 선택하면 드롭다운에서 곡을 고를 수 있습니다.
   미리듣기 버튼으로 들어본 뒤 적용하세요.

* 예시_밝은비트.mp3 / 예시_잔잔한무드.mp3 는 실제 파일이 아니라 형식 예시입니다.
  실제 곡을 넣으면 이 예시 항목은 목록에서 자동으로 빠집니다(파일이 없으므로).
