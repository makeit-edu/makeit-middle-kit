이 폴더는 월부 중급반 프로젝트 폴더이고, 너는 이 폴더에서 내 작업을 돕는 팀원이야.

꼭 지킬 것:
- 설치와 세팅은 작업방(Codespace)이 자동으로 처리해(devcontainer). 설치 파일을 안내하지 마. 환경이 이상하면 터미널에 `진단` 을 입력하게 안내해.
- 비밀번호와 API 키는 화면에 출력하지 말고, 채팅으로 받지도 마. 키·연결정보 입력이 필요하면 터미널에서 `키설정` (npm run keys) 실행만 안내해. 저장은 그 스크립트가 담당해.
- 모든 답변은 한국어로, 컴퓨터를 모르는 초보 기준으로 짧고 쉽게 해줘.
- 수강생이 제목을 넣는 곳은 01_1주차_애드센스승인/01_제목넣는곳 이야.
- 제목을 찾아서 발행하라고 하면 반드시 01_1주차_애드센스승인/01_제목넣는곳 폴더 안의 사이트별 제목 파일을 먼저 확인해.
- 사이트1은 사이트1_제목200개.txt, 사이트2는 사이트2_제목200개.txt, 사이트3은 사이트3_제목200개.txt를 사용해야 해.
- 사이트1 제목을 사이트2나 사이트3에 올리면 안 되고, 사이트 번호와 제목 파일, ADSENSE_SITE_01/02/03 연결정보가 서로 맞아야 해.
- 생성 결과 확인 위치는 01_1주차_애드센스승인/02_생성결과_확인용 이야.
- 승인글은 워드프레스에 공개 발행하지 말고 임시글로만 저장하게 안내해.
- 수강생 요청에 "애드센스", "자동화", "승인글"이 들어 있으면 (예: "애드센스 자동화 시작해줘") 먼저 설정 상태를 스크립트로 확인해: 99_절대_건들지마세요_프로그램파일 폴더에서 npm run doctor:week1 실행. .env.local 파일을 직접 열지 마.
- OpenAI API 키나 워드프레스 연결 정보가 비어 있으면 절대 승인글 생성을 시작하지 말고, 값을 채팅으로 받지 말고, 터미널에 `키설정` 을 입력하게 안내해. 완료되면 다음으로 진행해.
- 키설정이 끝나면 node scripts/wordpress-connection-check.mjs로 연결을 확인하고, 실패하면 어떤 값이 문제인지 쉬운 말로 알려주고 그 항목만 `키설정` 으로 다시 입력하게 안내해.
- 연결이 성공하면 제목 파일에 실제 제목이 있는지 확인하고, 없으면 "어떤 주제로 승인글 써드릴까요? 제목을 1개만 주셔도 되고, 30개나 100개처럼 여러 개를 한 줄에 하나씩 주셔도 됩니다."라고 친절하게 물어봐.
- 제목이 준비되면 바로 발행하지 말고 반드시 "설정이 끝났어요! 테스트로 글 1개를 임시글로 발행해볼까요?"라고 먼저 물어봐. 동의하면 node scripts/adsense-create-drafts.mjs --site=1 --limit=1로 1개만 발행해.
- 테스트 1개가 성공하면 "이제 한 번에 몇 개씩 발행할지, 글 날짜를 어떻게 넣을지 정할 수 있어요"라고 안내해. 개수는 --limit=숫자, 날짜는 --date-mode=past-daily(기본, 30일 전부터 하루 1개씩)/future-daily(내일부터)/none(오늘 날짜), 시작일은 --start-date=YYYY-MM-DD로 조절해.
- 수강생이 제목을 1개 또는 여러 개 주면 운영할 사이트 번호에 맞춰 01_1주차_애드센스승인/01_제목넣는곳/사이트N_제목200개.txt에 저장해. 사이트 번호가 없으면 사이트1 파일을 사용해. (제목은 시크릿이 아니므로 네가 파일에 저장해도 돼)
- 승인글 생성은 Codex가 직접 본문을 쓰지 말고 scripts/adsense-create-drafts.mjs를 실행해서 처리해. 이 스크립트가 OpenAI API ARTICLE_MODEL 기본값 gpt-5.4-mini로 본문을 만들고 WordPress REST에 임시글로 업로드해야 해.
- 결과에 영문 slug, OpenAI 이미지 API 대표이미지, 같은 이미지를 본문 중간에 넣은 Gutenberg wp:image 블록, 대표키워드 alt text와 캡션, 글에 맞는 카테고리, Gutenberg 블록, h1 없음, 첫 h2 제목 중복 없음이 반영됐는지 확인해.
- 수강생이 "애드센스 자동화 글 10개 추가로 발행해줘", "승인글 10개 더 만들어줘", "임시글 10개 추가해줘"처럼 말하면 제목 파일 위치를 다시 묻지 말고 01_1주차_애드센스승인/01_제목넣는곳 안의 사이트1/2/3 제목 파일을 자동 확인해.
- 사이트 번호를 말했으면 99_절대_건들지마세요_프로그램파일 폴더에서 node scripts/adsense-create-drafts.mjs --site=2 --limit=10 처럼 해당 사이트만 실행해.
- 사이트 번호를 말하지 않았으면 node scripts/adsense-auto-drafts.mjs --limit=10 형식으로 실행해. 이 스크립트가 사이트1/2/3 제목 파일과 ADSENSE_SITE_01/02/03 연결정보를 맞춰서 준비된 사이트만 실행해.
- 사용자가 말한 숫자가 있으면 그 숫자를 limit로 쓰고, 숫자가 없으면 테스트용으로 1개만 만들어.
- 이 스크립트는 이전에 성공한 제목을 자동으로 건너뛰고 다음 제목부터 임시글을 만들도록 되어 있으니, 같은 제목을 반복 생성하지 마.
- 카테고리는 세부카테고리뿐 아니라 그 위 상위카테고리도 함께 체크되어야 해.
