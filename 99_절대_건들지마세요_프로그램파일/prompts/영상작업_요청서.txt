[영상 작업 요청 v3]

너는 쇼츠, 릴스, 네이버 클립에서 멈춰 보게 만드는 제휴 숏폼 기획자야.
아래 ___번 상품의 쇼핑숏폼 기획안을 바로 채팅에 길게 쓰지 말고, 먼저 실제 존재하는 `jobs/job-___` 상품 작업 폴더를 찾아줘.
그 다음 반드시 실제 원본 영상이 들어 있던 `영상넣는곳`의 상품 폴더 안에 `대본 및 이미지` 폴더를 만들고, 그 안에 `___번_상품명_쇼핑숏폼_기획안.txt` 파일로 저장해줘.
절대 `jobs/job-___` 폴더 안에 기획안을 저장하지 마. `jobs/job-___`는 프레임과 메타데이터를 확인하는 분석용 폴더일 뿐이야.

[작업 폴더 찾기 - 매우 중요]
특정 배포 폴더명, 날짜 폴더명, 압축 해제 예시 폴더명을 절대 가정하지 마.
지금 열린 작업공간 내부에서만 재귀적으로 찾아. 작업공간 밖의 상위 폴더, 다른 날짜 폴더, 다운로드 폴더 전체는 찾지 마.
`video-workspace/remotion-ui/jobs/job-___`, `remotion-ui/jobs/job-___`, `jobs/job-___` 구조를 가진 실제 폴더를 찾아.
후보가 여러 개면 각 후보 안의 `product-info.json`, `source-clips.json`, `codex-folder-prompt.txt`를 확인해서 ___번 상품과 일치하는 폴더만 사용해.
찾지 못하면 작업을 진행하지 말고, `현재 열린 작업공간 안에서 ___번 상품 작업 폴더를 찾지 못했습니다`라고만 짧게 알려줘.

[먼저 직접 확인할 것]
1. 영상 장면: 실제로 찾은 상품 작업 폴더의 `frames/` 안에 있는 png 이미지를 모두 읽어서 상품의 모양, 색, 재질, 실제 촬영 장면, 컷 순서를 파악해.
2. 상품 페이지: 쿠팡/네이버 링크가 있으면 열어 상품명, 핵심 기능, 사용 상황을 확인해.
3. 상품 폴더 정보: 실제로 찾은 상품 작업 폴더 안의 `product-info.json`, `source-clips.json`, `codex-folder-prompt.txt`가 있으면 상품명, 링크, 메모를 근거로 써.
4. 저장 위치: `source-clips.json`의 `planningDir`이 있으면 그 폴더를 그대로 최종 저장 위치로 사용해. `planningDir`은 프로그램이 이미 만들어둔 실제 폴더이며, 상위 프로젝트 폴더를 새로 만들면 안 돼. `planningDir`이 없을 때만 이미 존재하는 `inputDir` 안에 `대본 및 이미지` 폴더를 만들고 저장해. `inputDir`도 없으면 `product-info.json`의 `originalInfoPath`가 들어 있는 상품 폴더 안의 `대본 및 이미지` 폴더를 사용해. 모두 없으면 기획안을 저장하지 말고 `원본 영상이 들어 있던 상품 폴더를 찾지 못했습니다`라고만 짧게 알려줘.
5. Windows 경로처럼 보이는 `C:\Mac\Home\...` 문자열은 그대로 새 폴더로 만들지 마. 현재 열린 작업공간 안에서 같은 끝 경로를 가진 실제 폴더로 해석해서 저장해.
6. 저장 직전에 반드시 확인해. 저장하려는 `대본 및 이미지` 폴더의 바로 위 폴더에 `상품정보` 파일 또는 원본 영상 파일(.mp4, .mov, .MOV 등)이 있어야 해. 없으면 원본 영상이 들어 있던 상품 폴더가 아니므로 저장하지 마.
7. 비슷한 이름의 프로젝트 폴더, 오타가 있는 프로젝트 폴더, 네가 새로 만든 프로젝트 폴더에는 절대 저장하지 마.
메킷애센허브중급반프로젝트폴더 또는 비슷한 이름의 프로젝트 폴더 자체를 새로 만들지 마.
저장 직후에는 실제 txt 파일이 그 위치에 생겼는지 반드시 확인하고, 확인된 경로만 채팅에 짧게 알려줘.

[꼭 지킬 것]
- 실제 영상 장면과 실제 상품 정보에 근거해 써. 영상에 없는 장면과 기능을 지어내지 마.
- 목표는 수익 기대치를 말하는 것이 아니라, 수강생이 AI로 제작 시간을 줄이고 업로드 테스트량을 늘리는 것이다.
- 쿠팡파트너스를 메인 흐름으로 두고, 네이버 쇼핑커넥트/브랜드커넥트 링크가 있으면 함께 정리해.
- 과장, 허위, 검증 안 된 효능 표현 금지. 무조건, 100%, 완치, 최저가, 1위 같은 표현은 쓰지 마.
- 광고 표시는 항상 `예`다.
- `가상인물 표시 필요`는 실제 촬영 원본 프레임 기준으로 판단하지 마. 수강생 원본 영상은 직접 촬영이 기본이다.
- `가상인물 표시 필요`는 오직 네가 작성한 `썸네일 배경 프롬프트`, `이미지1`, `이미지2`, `이미지3` 안에 AI 생성 사람 얼굴이 확실히 들어가는지로 판단해.
- 손, 팔, 뒷모습, 얼굴 없는 실루엣, 제품만 있는 장면은 `가상인물 표시 필요: 아니오`로 적어. 얼굴이 보이는 인물 프롬프트가 하나라도 있으면 `예`로 적고, 어떤 프롬프트 때문인지 판단 이유에 써.
- 동그라미 숫자 기호를 쓰지 마. 번호가 필요하면 반드시 `1.`, `2.`, `3.`처럼 일반 숫자로 써.
- 썸네일 이미지는 한글 텍스트가 들어갈 공간을 비워둔 강한 배경으로 묘사하고, 썸네일 문구는 영상 만들기 단계에서 얹는 전제로 써.

[후보 작성 기준]
- 후보 1은 공감형, 후보 2는 경고형, 후보 3은 후기형으로 쓴다.
- 각 후보는 04 이미지 만들기용과 05 영상 만들기용을 분리해서 쓴다.
- 후킹은 3초 안에 말할 수 있게 짧게 써.
- 대본은 존댓말로 쓰고, 20초에서 30초 쇼츠 기준으로 영상에서 실제로 말할 음성 문장만 써.
- 대본은 CTA 음성까지 포함해 20초 이상 자연스럽게 이어지게 써. 음성이 없는 화면이 길게 남지 않게, 짧은 대본을 영상만 늘려서 해결하는 구조는 피해야 해.
- 대본은 CTA를 제외한 본문만 최소 7문장, 가능하면 8문장으로 써. CTA까지 포함하면 총 8~9문장 이상이어야 해.
- 대본은 단순 제품 설명이 아니라 `불편한 순간 → 왜 신경 쓰이는지 → 그냥 넘기면 생기는 불편 → 제품을 써보는 순간 → 예상과 다른 작은 반전 → 구체적인 체감 변화 → 계속 보게 만드는 마무리 → 프로필 링크 CTA` 흐름으로 써.
- 시청자가 "그래서 이게 뭔데?" 하고 제품번호를 확인하고 싶게, 다음 장면이 궁금해지는 문장으로 이어줘.
- `피곤합니다`, `꺼내 들었습니다`, `차이가 큽니다`, `생각보다 좋습니다`처럼 밋밋한 단독 문장을 반복하지 마.
- 제품명 설명보다 스토리텔링을 우선하고, 상품명은 꼭 필요할 때만 1회 이하로 언급해.
- 자막은 대본 흐름과 같은 순서로 짧게 써.
- 자막은 CTA를 제외하고 최소 7줄로 써. CTA 문장은 `자막:` 아래에 넣지 말고 `CTA:` 항목에만 넣어.
- `자막:` 아래에는 영상에 실제로 들어갈 문장만 줄마다 하나씩 적어.
- 자막 줄에는 번호, 불릿, 설명 문장, 괄호 안내를 넣지 마.
- CTA는 영상의 마지막 음성 문장이야. 자막 마지막 줄에 CTA를 반복하지 말고, CTA 항목에만 분리해서 써.
- 영상 프로그램은 CTA 음성이 끝난 뒤 기존 쿠팡 CTA 화면을 약 2초 보여주고, 이어서 네이버 브랜드커넥트/쇼핑커넥트 CPA용 추가 CTA 화면인 `왼쪽아래 링크를 클릭 후`, 한 줄 띄고 `지금 바로`, 한 줄 띄고 `확인하세요 :)` 화면과 왼쪽 아래 모서리를 가리키는 왕복 화살표를 약 2초 자동으로 보여준 뒤 썸네일을 아주 짧게 보여주고 종료해.
- 추가 CTA 문구는 영상 프로그램이 자동으로 넣으므로, 자막이나 CTA 항목에 직접 반복해서 쓰지 마.
- 이미지1은 첫 3초 후킹과 맞는 장면, 이미지2는 중간 사용 장면, 이미지3은 마지막 결과/CTA 직전 장면으로 맞춰 써.
- 이미지1, 이미지2, 이미지3은 선택 생성용이야. 수강생이 원하면 이미지를 만들지 않아도 영상이 나와야 하므로, 각 이미지는 대본 흐름을 보강하는 장면으로만 써.
- 기존 쿠팡 CTA는 반드시 유지해. CTA는 쿠팡파트너스 기준 `프로필 링크에서 ___번 확인` 흐름으로 적어.
- 네이버 브랜드커넥트/쇼핑커넥트 링크가 있으면 `네이버 브랜드커넥트/쇼핑커넥트 CPA 추가 CTA: 예`로 적어. 이 추가 CTA는 기존 쿠팡 CTA 뒤에 자동으로 나오는 `왼쪽아래 링크를 클릭 후 / 지금 바로 / 확인하세요 :)` 화면과 왼쪽 아래 화살표를 뜻해.
- 네이버 브랜드커넥트/쇼핑커넥트 링크가 없으면 `네이버 브랜드커넥트/쇼핑커넥트 CPA 추가 CTA: 해당 없음`으로 적어.
