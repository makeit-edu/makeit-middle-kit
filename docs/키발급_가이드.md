# API 키 발급 가이드 (처음 하시는 분용)

키트를 돌리려면 **키 2개**가 필요합니다. 이 문서 하나로 두 키를 모두 발급받고, 마지막에 `키설정` 으로 입력하는 것까지 끝냅니다. 소요 시간은 두 키 합쳐 15~20분입니다.

## 이 키 2개가 뭔가요?

| 키 | 하는 일 | 발급처 | 비용 |
|---|---|---|---|
| ① OpenAI API 키 | 글·이미지를 만드는 공장 열쇠 | platform.openai.com | 쓴 만큼 소액 결제 (선불 충전) |
| ② ElevenLabs API 키 | 영상 음성(내레이션)을 만드는 공장 열쇠 | elevenlabs.io | 무료로 시작 가능 |

> **"Codex 로그인을 했는데 왜 키가 또 필요해요?"**
> Codex 로그인(온보딩 4단계)은 "대화 비서의 열쇠"로, ChatGPT Plus 구독에 포함되어 추가 비용이 없습니다.
> 이 키 2개는 "글·이미지·음성 공장을 돌리는 열쇠"로, 사용한 만큼만 별도로 소액 과금됩니다. 서로 다른 열쇠입니다.

발급 화면은 모두 **영어**입니다. 아래 순서 그대로 따라오시면 됩니다. 준비물: 이메일 주소, 해외 결제 가능한 카드(비자/마스터 — OpenAI용).

---

## ① OpenAI API 키 발급 (글·이미지용)

**주의: chatgpt.com이 아니라 platform.openai.com 입니다.** 같은 회사의 다른 사이트예요.

### 1. 가입/로그인

1. 크롬에서 **platform.openai.com** 접속.
2. 우측 상단 **Log in**(로그인) 클릭 → ChatGPT에서 쓰는 계정 그대로 로그인하면 됩니다. 계정이 없으면 **Sign up**(가입).
   [스크린샷: platform.openai.com 첫 화면의 Log in 버튼 위치]

### 2. 결제수단 등록 (필수)

OpenAI API는 **선불 충전 방식**이라, 카드를 등록하고 소액을 충전해야 키가 작동합니다.

1. 우측 상단 **톱니바퀴(Settings)** → 왼쪽 메뉴에서 **Billing**(결제) 클릭.
   [스크린샷: Settings → Billing 메뉴 위치]
2. **Add payment details**(결제수단 추가) 버튼 → **Individual**(개인) 선택 → 카드 번호 입력.
3. 충전 금액을 물어보면 **$10** 를 권장합니다(최소 $5). "자동 충전(Auto recharge)"은 **꺼두세요** — 모르는 사이 결제되는 것을 막기 위해서입니다.
   [스크린샷: 충전 금액 입력 화면, Auto recharge 끔 상태]

> 비용 감: 승인글 1개(본문+이미지)에 몇십 원~백 원 수준입니다. $10이면 1주차 실습을 여유 있게 진행합니다.

### 3. 키 만들기

1. 왼쪽 메뉴(또는 상단)에서 **API keys** 클릭.
2. **Create new secret key**(새 비밀 키 만들기) 버튼 클릭.
   [스크린샷: API keys 화면의 Create new secret key 버튼]
3. Name(이름)에는 아무거나 입력해도 됩니다. 예: `weolbu`. 나머지는 그대로 두고 **Create secret key**.
4. `sk-` 로 시작하는 긴 글자가 나타납니다. **지금 바로 Copy(복사) 버튼을 누르고 메모장에 붙여넣어 두세요.**
   [스크린샷: 생성된 키와 Copy 버튼]

> **가장 중요: 이 창을 닫으면 키를 다시 볼 수 없습니다.** 복사를 놓쳤다면 당황하지 말고 그 키는 버리고 새로 하나 만들면 됩니다(몇 개를 만들어도 무료).

## ② ElevenLabs API 키 발급 (음성용)

### 1. 가입

1. 크롬에서 **elevenlabs.io** 접속 → 우측 상단 **Sign up**(가입).
2. 구글 계정으로 가입하면 가장 빠릅니다 (**Continue with Google**).
   [스크린샷: ElevenLabs 가입 화면]
3. 플랜을 물어보면 **Free**(무료)로 시작하세요. 무료로도 영상 몇 개는 만들 수 있고, 부족해지면 그때 Starter($5/월)로 올리면 됩니다.

### 2. 키 만들기

1. 로그인 후 **좌측 하단의 내 계정(프로필) 아이콘** 클릭 → **API Keys** 메뉴 클릭.
   [스크린샷: 프로필 메뉴에서 API Keys 위치]
2. **Create API Key**(키 만들기) 버튼 → 이름은 아무거나(예: `weolbu`) → 만들기.
3. 생성된 키를 **Copy(복사)** 해서 메모장에 붙여넣어 두세요. 이 키도 창을 닫으면 다시 볼 수 없습니다.
   [스크린샷: 생성된 키와 복사 버튼]

## ③ 발급받은 키를 키트에 입력하기

1. 작업방(Codespace)으로 돌아와서, 화면 아래 **터미널**에 `키설정` 을 입력하고 Enter.
2. 질문이 아래 순서로 나옵니다. 순서대로 붙여넣으면 됩니다:
   - **① 수강 코드** (강의 자료실 공지에 있습니다)
   - **② 주차 선택** — `[1] 1주차 / [2] 2주차 / 엔터 = 전부` 중 하나. 잘 모르면 그냥 **엔터**.
   - **③ OpenAI 키**
   - **④ (1주차) 워드프레스 사이트 1~3의 도메인·관리자 ID·애플리케이션 비밀번호** — **아직 준비 전이면 엔터로 건너뛰세요.**
   - **⑤ ElevenLabs 키**
   - **⑥ (2주차) 쇼핑숏폼 워드프레스 정보** — 준비 전이면 엔터로 건너뛰세요.
   - 붙여넣어도 **화면에 안 보이는 게 정상**입니다(보안). 한 번만 붙여넣고 Enter.
   - **준비 안 된 항목은 엔터로 건너뛰면 됩니다.** 나중에 `키설정` 을 다시 실행하면 그 값만 넣을 수 있어요.
3. 화면에 "**3개 모두 정상 확인**"이 나오면 끝입니다. (건너뛴 항목은 [확인 필요]/[나중에 입력]으로 표시돼요 — 실패가 아닙니다)
4. [확인 필요]가 뜬 항목이 있으면 → 그 키만 발급 페이지에서 다시 복사해 `키설정` 을 다시 실행하세요.

## 꼭 지켜주세요

- **키는 비밀번호입니다.** 채팅방·카톡·인증샷에 절대 올리지 마세요.
- 메모장에 옮겨둔 키는 `키설정` 입력이 끝나면 지우는 것이 안전합니다.
- 키가 유출된 것 같으면 → 발급 페이지에서 그 키를 삭제(Revoke)하고 새로 만들어 `키설정` 을 다시 하면 됩니다. 코치에게 알려주시면 도와드립니다.
