# 1주차 사용방법 — 애드센스 승인글 자동화

1주차는 메킷애센 플러그인으로 애드센스 승인용 사이트 구조를 세팅하고, Codex가 OpenAI로 승인글 본문과 대표이미지를 만들어 내 워드프레스에 **임시글**로 올려주는 과정입니다.

> 이 문서는 브라우저 작업방(Codespace) 기준입니다. 내 컴퓨터에 설치하는 것이 없고, 더블클릭할 실행 파일도 없습니다. 모든 명령은 **화면 아래 터미널(검은 창)** 에 입력합니다.

---

## 1. 내가 직접 하는 것

1. **워드프레스 사이트 3개**를 준비합니다.
2. 각 워드프레스에 **메킷애센 middle_class 플러그인**을 설치합니다. (플러그인 파일은 강의 자료실에서 받습니다)
3. **사이트별 제목 200개**를 GPTs에서 뽑아 `01_제목넣는곳` 폴더의 파일에 넣습니다.
   - 왼쪽 탐색기에서 `01_1주차_애드센스승인/01_제목넣는곳/사이트1_제목200개.txt` 클릭 → 가운데 화면에 열리면 제목을 붙여넣기 → **Ctrl+S (맥은 Cmd+S)** 로 저장.
     [스크린샷: 탐색기에서 제목 파일을 열어 붙여넣는 화면]
   - 파일을 컴퓨터에서 만들어 올릴 필요 없이, 이렇게 화면에서 바로 붙여넣으면 됩니다.
   - 파일명에 사이트 구분만 들어가면 Codex가 알아서 찾습니다. 예: `사이트2_제목200개.txt`, `두번째사이트_제목.txt`
   - 가장 안전한 형태는 **한 줄에 제목 하나씩**입니다. GPTs 출력처럼 `1. 제목` 번호가 붙어 있어도 자동으로 제목만 추출합니다.
   - **파일 안에 사이트 주소, 아이디, 비밀번호, API 키는 절대 넣지 마세요.**
4. 터미널에 `시작` 을 입력해 Codex 대화창을 열고, `애드센스 승인글 자동화 시작해줘` 라고 입력합니다.

## 2. 연결 정보는 채팅이 아니라 `키설정`으로

예전과 달리, **워드프레스 주소·아이디·애플리케이션 비밀번호·API 키를 Codex 채팅창에 쓰지 않습니다.**
Codex가 연결 정보가 필요하다고 하면 터미널에 `키설정` 을 입력하고, 안내에 따라 붙여넣으면 됩니다.

- 키 발급 방법을 모르면 → `docs/키발급_가이드.md` 를 보세요.
- 붙여넣어도 화면에 안 보이는 게 정상입니다(보안). 한 번만 붙여넣고 Enter.
- 키는 비밀번호입니다. 채팅방·카톡에 절대 올리지 마세요.

## 3. Codex(프로그램)가 대신 하는 것

- 워드프레스 3개 연결을 확인합니다.
- 제목 파일에서 제목을 읽고, 제목마다 영어 주소(슬러그)를 자동으로 만듭니다.
- 승인글 프롬프트 원본을 기준으로 본문을 생성합니다.
- 제목과 어울리는 16:9 가로형 대표이미지를 생성합니다.
- 워드프레스에 **임시글**로 올리고 대표이미지까지 설정합니다.
- 생성 결과와 비용 추정치를 `02_생성결과_확인용` 폴더에 저장합니다.

## 4. 글 생성 순서 (꼭 이 순서대로)

1. 처음에는 무조건 **테스트 1개**만 만듭니다. 연결과 글 품질을 확인합니다.
2. 괜찮으면 **5개** → 그다음부터 **20개 단위**로 늘립니다.
3. 사이트 1개당 **임시글 100개 이상**을 채운 뒤 애드센스 승인을 신청하는 것이 목표입니다. 제목 200개를 준비해두면 승인 신청 후에도 글을 계속 추가할 수 있습니다.

## 5. 중요한 규칙

- 승인글은 **임시글로만** 저장합니다. 공개 발행하지 않습니다.
- 제목은 제목 파일에 있는 것을 그대로 사용합니다. Codex가 마음대로 만들지 않습니다.
- 대표이미지는 기본 1536x864(16:9)이고, 이미지 안에 글자를 넣지 않습니다.
- 요약글은 워드프레스에서 직접 작성합니다.
- 글 날짜는 기본적으로 **과거 날짜에 하루 1개씩** 자동 분산됩니다. 미래 예약을 원하면 Codex에게 말하세요.

- 뭔가 이상하면 터미널에 `진단` → 출력 전체를 복사해 문의 채널에 올려주세요.
