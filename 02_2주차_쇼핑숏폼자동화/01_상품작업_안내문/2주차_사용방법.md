# 2주차 사용방법 — 쇼핑숏폼 영상 자동화

2주차는 직접 촬영한 상품 영상을 올리면, 프로그램이 기획안 → 이미지 → 음성 → 완성 쇼핑숏폼까지 만들어주는 과정입니다. 쿠팡파트너스 중심 제휴 숏폼의 제작 시간을 줄이고 업로드 테스트량을 늘리는 것이 목표입니다.

> 이 문서는 브라우저 작업방(Codespace) 기준입니다. 더블클릭할 실행 파일(.command/.bat)은 이제 없습니다. 모든 명령은 **화면 아래 터미널(검은 창)** 에 한 단어로 입력합니다.

---

## 0. (준비) 휴대폰 영상을 컴퓨터로 옮기기

원본 영상은 대부분 휴대폰에 있습니다. 먼저 컴퓨터로 옮기세요. 쉬운 순서대로 3가지 방법:

1. **카톡 "나에게 보내기"** → 컴퓨터 카톡(PC버전)에서 저장 — 가장 쉽습니다.
2. USB 케이블로 휴대폰을 연결해 복사.
3. 구글 포토 등 클라우드에 올렸다가 컴퓨터에서 내려받기.

옮기면서 **파일 이름을 `1.mp4, 2.mp4, 3.mp4...` 숫자로 바꿔두세요.** (한글 파일명은 오류의 원인이 됩니다. 숫자 순서대로 영상이 이어붙습니다)

## 1. 영상 올리기 (업로드)

1. 왼쪽 탐색기에서 `02_2주차_쇼핑숏폼자동화/영상넣는곳/` 폴더에 **상품 폴더**를 만듭니다.
   - 폴더 이름: `상품번호_상품명` 형식. 예: `001_손목보호대`, `002_주방가위`
   - 상품번호는 항상 3자리 숫자입니다: `001`, `002`, `003`...
   - 만드는 법: `영상넣는곳` 폴더에서 우클릭 → New Folder(새 폴더).
2. 컴퓨터에 옮겨둔 mp4 파일들을 **마우스로 끌어다** 그 폴더에 놓으면 업로드됩니다. (또는 폴더 우클릭 → Upload...)
   [스크린샷: 탐색기의 상품 폴더에 mp4 파일을 드래그해 업로드하는 장면]
   - 영상 파일은 커서 몇 분 걸릴 수 있습니다. **업로드가 끝날 때까지 브라우저 탭을 닫지 마세요.**
3. 같은 폴더 안에 `상품정보.txt` 파일을 만들어 화면에서 직접 작성합니다. (우클릭 → New File)
   - 넣을 내용: 상품명, 쿠팡파트너스 링크, (있다면) 네이버 링크, 참고 메모.
   - 다 쓰면 **Ctrl+S (맥은 Cmd+S)** 로 저장.

완성 예시:

```
영상넣는곳/
└── 001_손목보호대/
    ├── 1.mp4
    ├── 2.mp4
    └── 상품정보.txt
```

## 2. 새 상품 작업 만들기

터미널에 `새상품 1` 처럼 **상품번호를 붙여** 입력합니다. (001번 상품이면 `새상품 1`)

- 프로그램이 원본 영상을 하나로 합치고, Codex가 볼 장면(프레임)을 뽑아 작업 폴더를 만듭니다.
- "완료: N번 작업이 만들어졌습니다"가 나오면 성공입니다.

## 3. 기획안 만들기 (Codex)

1. 터미널에 `시작` 을 입력해 Codex 대화창을 엽니다.
2. `영상 만들어줘` 또는 `1번 상품 기획안 만들어줘` 라고 입력합니다.
3. Codex가 실제 촬영 장면과 상품정보를 읽고 기획안을 만들어, **상품 폴더 바로 아래**에 `001_상품명_기획안.txt` 로 저장합니다.
4. 왼쪽 탐색기에서 그 파일을 열어 후보 3개(공감형/경고형/후기형) 중 마음에 드는 것을 확인하세요.

## 4. 이미지 만들기

터미널에 `이미지만들기` 를 입력합니다. 기획안의 이미지 문구로 썸네일·보조 이미지를 생성합니다. (이미지 없이 영상만 만들어도 됩니다 — 선택 단계)

## 5. 영상 만들기 (렌더)

터미널에 `영상만들기` 를 입력합니다.

- 음성(TTS)을 만들고 영상을 조립합니다. **몇 분 걸립니다 — 진행 표시가 움직이면 정상입니다.**
- 완성본은 `02_2주차_쇼핑숏폼자동화/완성영상/` 폴더에 생깁니다. (쿠팡용/네이버용 2가지)

## 6. 편집기로 다듬기 (선택)

터미널에 `편집기` 를 입력합니다.

- 잠시 후 화면 안에 미리보기 창이 열리거나, 우측 아래에 **"포트 4321"** 알림이 뜹니다 → "브라우저에서 열기"를 누르세요.
  [스크린샷: 포트 4321 알림과 열기 버튼]
- 예전처럼 주소창에 localhost를 입력하는 것이 아니라, 이 **알림의 주소**로 들어가는 것만 다릅니다. 기능은 같습니다.
- 배경음악을 바꾸고 싶으면: `99_절대_건들지마세요_프로그램파일/video-workspace/remotion-ui/public/bgm/` 폴더에 mp3를 끌어다 놓으세요. (이 폴더는 예외적으로 만져도 됩니다)

## 7. 완성 영상 내려받기

1. 왼쪽 탐색기에서 `완성영상/` 안의 파일을 **우클릭 → Download...** 하면 내 컴퓨터로 내려옵니다.
2. 여러 개를 한 번에 받으려면 터미널에 `내보내기` → zip 파일 하나로 묶어줍니다.
3. **내려받은 파일은 브라우저 다운로드 폴더에 있습니다.** 크롬 우상단 화살표(↓) → "폴더에서 보기" → 바탕화면의 '완성영상' 폴더로 옮겨두세요.
   [스크린샷: 크롬 다운로드 목록에서 "폴더에서 보기" 위치]

> **완성 영상은 만들면 바로 내려받으세요.** 이 작업방은 작업대이지 보관함이 아닙니다. 30일 이상 접속하지 않으면 작업방이 삭제됩니다.

## 8. 뒷정리

다운로드가 끝나면 터미널에 `정리` 를 입력하세요. 끝난 작업의 중간 파일을 지워 공간을 확보합니다. (지우기 전에 완성영상을 내려받았는지 물어봅니다)

마지막으로 `저장` 을 입력해 오늘 작업(상품정보·기획안 등 텍스트)을 클라우드에 기록하세요.

---

## 막히면

- 터미널에 `진단` 입력 → 출력 전체를 복사해 문의 채널에 올려주세요.
- 영상이 안 올라가요 → 파일명이 `1.mp4` 처럼 숫자인지, 업로드 중 탭을 닫지 않았는지 확인.
- 완성영상이 안 보여요 → `영상만들기` 가 끝까지 진행됐는지 확인 후 `진단`.
- 다운받은 영상이 어디 갔는지 모르겠어요 → 브라우저 다운로드 폴더에 있습니다 (위 7번).
